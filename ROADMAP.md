### 🔧 已知问题

#### 可选功能失败 (7/692 测试)
1. **ONNX Export (4 个测试)** - 需要安装可选依赖 `onnxscript`
   - 影响：ONNX 模型导出功能不可用
   - 解决方案：`pip install onnxscript` 或跳过 ONNX 导出

2. **TorchScript (3 个测试)** - Python 3.14 不支持 TorchScript（已弃用）
   - 影响：TorchScript 模型导出功能不可用
   - 解决方案：使用 `torch.export` 或降级到 Python 3.11

#### 建议
- 核心功能完全可用，上述问题仅影响可选的模型导出功能
- 推荐使用 PyTorch 原生格式 (.pt) 或 ONNX (安装依赖后)

3. **优先级**: 是否比其他需求更紧急？
